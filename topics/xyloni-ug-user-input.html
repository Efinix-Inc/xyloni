# User Inputs The board has 2 pushbutton switches that you can use as inputs to the T8F81C2 device. The T8F81C2 bank 2A I/O signals connected to these switches have a pull-up resistor. When you press the switch, the signal drives low, indicating user input. |Reference Designator|T8F81C2 Pin Name|Active| |:------------------:|:--------------:|:----:| |BTN1|GPIOR\_02\_RESERVED\_OUT|Low| |BTN2|GPIOR\_15\_CBUS0|Low| **Parent topic:**[Board Functional Description](../topics/xyloni-ug-fd.html)
